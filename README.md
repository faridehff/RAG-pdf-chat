#  AI PDF Chat (Local RAG with Ollama)

This project allows users to upload a PDF and ask questions about its content using a local LLM.

##  Features

* PDF ingestion
* Semantic search (FAISS)
* Local LLM (Ollama - Llama3)
* Streamlit UI

##  Tech Stack

* Python
* FAISS
* Sentence Transformers
* Streamlit
* Ollama (local LLM)

##  Run

```bash
streamlit run app_ui.py
```

##  Notes

* Runs fully locally (no API cost)
* Demonstrates Retrieval-Augmented Generation (RAG)
